## Instrukcja uruchomienia
1. Sklonuj repozytorium
```bash
git clone https://github.com/S1iwa/Temp-python.git
```
2. Zainstaluj wymagane biblioteki
```bash
pip install -r requirements.txt
```
3. Uruchom program
```bash
python src/main.py
```
